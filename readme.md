## 🇬🇧 About
**Discord Token Viewer** is a secure and **open-source** browser extension built for Discord developers, bot owners, and curious users.  
It never sends your token anywhere; it just **displays** it locally. 🔍✨

### 🛡️ Security Features
- 📦 **Fully open source:** Check and audit the code anytime.
- 🚫 **No logging:** Never transmits your data or tokens externally.
- 🧩 **Just displays:** Lets you view your token safely in your browser.

### ⚙️ Features
✅ Simple and user-friendly interface  
✅ Doesn’t ask for unnecessary permissions  
✅ Lightweight, fast, and free
